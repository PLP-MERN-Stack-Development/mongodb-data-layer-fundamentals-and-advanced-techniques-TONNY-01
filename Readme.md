# MongoDB Query Assignment

This repository contains solutions and practice exercises for the **MongoDB Data Layer Fundamentals and Advanced Techniques** module.  
The goal is to strengthen understanding of CRUD operations, filtering, indexing, and replica sets in MongoDB.

---

##  Contents
- Basic queries using `find()`
- Insert operations with `insertOne()` and `insertMany()`
- Update operations with `updateOne()` and `updateMany()`
- Delete operations with `deleteOne()` and `deleteMany()`
- Filtering with operators like `$and`, `$or`, and `$in`
- Indexing for query optimization
- Introduction to replica sets for high availability
